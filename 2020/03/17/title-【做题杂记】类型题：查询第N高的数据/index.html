使用 limit 和 offset
limit n子句表示查询结果返回前n条数据

offset n表示跳过x条语句

limit y offset x 分句表示查询结果跳过 x 条数据，读取前 y 条数据

注意：limit x：y和offset不同
前者表示跳过x条数据读取y条数据
